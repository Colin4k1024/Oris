# Evo Shareable Reference Assets

This directory is reserved for Evo benchmark shareable assets in the `evo_oris_repo` example suite.

The benchmark scenario creates patch targets under these subdirectories:

- `docs/`
- `code/`
- `config/`

Keeping these assets under one root makes cross-agent share/replay examples easier to inspect and avoids mixing benchmark artifacts with project docs or source modules.
